# JWT-Basics

    API for authentication using JWT

## Install

npm install

## Running

npm start

## Base url

### `http://localhost:3000`

## Routes

### router.post("/api/v1/login");

### router.get("/api/v1/dashboard");

## Note

### Add a `.env `file to the root of your project with the following content:

` JWT_SECRET = <secret code>`
